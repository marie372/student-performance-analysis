STUDENT PERFORMANCE ANALYSIS PROJECT

Project Overview:
This project analyzes student academic performance using SQL, Python, and Power BI.

Tools Used:
- PostgreSQL
- Python
- pandas
- matplotlib
- Power BI
- Excel

Project Objectives:
- Analyze student exam performance
- Identify factors affecting academic success
- Create interactive dashboards
- Generate business insights from educational data

Key Insights:
- Higher study hours improve exam performance
- Attendance positively affects scores
- Higher stress levels reduce academic performance
- Motivation influences student success

Files Included:
- student_performance_clean.csv
- analysis.py
- queries.sql
- student_performance_dashboard.pbix
